# Collection Manipulator

## Project Description
Collection Manipulator is a Python-based project that helps users perform various operations on a collection of student records. The project uses a list of dictionaries to store and manage student information.

## Features
1. Add Student Data
2. Display All Students
3. Search Student by ID
4. Update Student Information
5. Delete Student Record
6. Exit Program

## Technologies Used
- Python 3
- List
- Dictionary
- Loops
- Conditional Statements

## How to Run
1. Open the Python file.
2. Run the program.
3. Select an option from the menu.
4. Follow the instructions displayed on the screen.
